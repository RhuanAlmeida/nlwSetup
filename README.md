## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e GitHub
- Figma

## Projeto

Esse projeto é um controlador/registrador de hábitos, onde você registra no dia a dia hábitos que foram executados ou não.
